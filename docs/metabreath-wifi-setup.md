# วิธีเชื่อมต่อ MetaBreath กับ WiFi (สำหรับผู้ใช้ใหม่)

คู่มือนี้ใช้สำหรับการตั้งค่าครั้งแรก หรือเมื่อต้องการเปลี่ยน WiFi ใหม่

---

## สิ่งที่ต้องมีก่อนเริ่ม

- อุปกรณ์ MetaBreath (เปิดอยู่และมีไฟ)
- โทรศัพท์มือถือ (Android หรือ iPhone)
- ชื่อ WiFi (SSID) และรหัสผ่าน WiFi ที่บ้านหรือสถานที่ใช้งาน
- Device ID ของคุณ (ดูวิธีหาในขั้นตอนที่ 1)

> **หมายเหตุ:** WiFi ที่ใช้ต้องไม่บล็อก port 1883 (WiFi สาธารณะ เช่น ห้างสรรพสินค้า หรือเน็ตมือถือ AIS มักบล็อก port นี้)  
> แนะนำ: WiFi บ้าน หรือ hotspot Android ที่ไม่ใช่ AIS

---

## ขั้นตอนที่ 1 — หา Device ID จาก Web App

1. เปิด [metabreath.duckdns.org](https://metabreath.duckdns.org) บนโทรศัพท์หรือคอม
2. เข้าสู่ระบบด้วยบัญชีของคุณ
3. ไปที่หน้า **Me → Device**
4. กดเมนู **"Download firmware (.ino)"**
5. ดู URL ในแถบที่อยู่ — จะเห็นตัวเลขยาวๆ รูปแบบนี้:

```
metabreath.duckdns.org/me/device/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx/firmware
```

6. **คัดลอก UUID** (ส่วนที่เป็น `xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx`) เก็บไว้ใช้ในขั้นตอนที่ 4

---

## ขั้นตอนที่ 2 — เชื่อมต่อกับ MetaBreath Portal

1. เปิดการตั้งค่า WiFi บนโทรศัพท์
2. มองหา WiFi ที่ชื่อขึ้นต้นด้วย **`MetaBreath-Setup-`** (เช่น `MetaBreath-Setup-2810`)
3. กดเชื่อมต่อ — ไม่ต้องใส่รหัสผ่าน

> **iPhone:** หลังเชื่อมอาจมี popup เด้งขึ้นมา **ให้ปิด popup นั้น** แล้วเปิด Safari ด้วยตัวเอง

---

## ขั้นตอนที่ 3 — เปิดหน้าตั้งค่า

- เปิดแอป **Safari** (iPhone) หรือ **Chrome** (Android)
- พิมพ์ในแถบที่อยู่: **`192.168.4.1`** แล้วกด Enter
- หน้าเว็บ MetaBreath Setup จะเปิดขึ้น

---

## ขั้นตอนที่ 4 — กรอกข้อมูล WiFi และ Device ID

1. กด **"Configure WiFi"**
2. รายชื่อ WiFi จะแสดงขึ้น — **เลือก WiFi บ้านของคุณ**
3. กรอก **รหัสผ่าน WiFi**
4. ในช่อง **"Device ID"** — กรอก UUID ที่คัดลอกมาจากขั้นตอนที่ 1
5. กด **"Save"**

อุปกรณ์จะบันทึกข้อมูลและ restart อัตโนมัติ

---

## ขั้นตอนที่ 5 — ตรวจสอบว่าเชื่อมต่อสำเร็จ

1. โทรศัพท์จะกลับไปใช้ WiFi ปกติโดยอัตโนมัติ
2. รอประมาณ **30 วินาที**
3. กลับไปที่ web app → หน้า Device
4. สถานะจะเปลี่ยนเป็น **"Live · กำลังส่งข้อมูล"** (จุดเขียวกะพริบ)
5. ข้อมูลจะเริ่มแสดงใน Recent Readings

---

## จับคู่ใหม่ (Re-pair) เมื่อต้องการเปลี่ยน WiFi

ถ้าต้องการเปลี่ยน WiFi หรืออุปกรณ์ไม่เชื่อมต่อ:

1. **ถอดสาย USB** (ปิดไฟ) แล้วเสียบใหม่
2. ถ้า WiFi เดิมยังใช้ได้ — อุปกรณ์จะเชื่อมต่ออัตโนมัติ ไม่ต้องทำอะไร
3. ถ้า WiFi เดิมใช้ไม่ได้ (หรือลืมรหัส) — อุปกรณ์จะเปิด Portal อัตโนมัติภายใน 30 วินาที
4. ทำตามขั้นตอนที่ 2–4 ซ้ำอีกครั้ง

---

## เปลี่ยน user (ส่งอุปกรณ์ให้คนอื่น)

กรณีนำ MetaBreath ไปใช้กับบัญชี MetaBreath อีกบัญชีหนึ่ง ต้องล้างข้อมูลเก่าออกก่อน:

### วิธีล้างและตั้งค่าใหม่

ต้องใช้คอมพิวเตอร์ที่มีสาย USB และติดตั้ง [esptool](https://pypi.org/project/esptool/) ไว้แล้ว

**ขั้นตอน:**

1. เสียบสาย USB เชื่อมต่อ MetaBreath กับคอม
2. เปิด Terminal แล้วรันคำสั่งนี้ (แทน `PORT` ด้วย port จริง เช่น `/dev/cu.usbserial-0001` หรือ `COM3`):
```bash
esptool --port PORT erase_region 0x9000 0x5000
```
3. ถอดและเสียบสาย USB ใหม่ — อุปกรณ์จะเปิด Portal ทันที
4. ทำตามขั้นตอนที่ 2–4 โดยกรอก UUID ของ user ใหม่

---

## ปัญหาที่พบบ่อย

| ปัญหา | วิธีแก้ |
|---|---|
| ไม่เห็น WiFi `MetaBreath-Setup-` | ตรวจสอบว่าอุปกรณ์เปิดอยู่ รอ 30 วิ แล้วรีเฟรช WiFi |
| กด popup แล้วหน้าเว็บไม่เปิด (iPhone) | ปิด popup → เปิด Safari → พิมพ์ `192.168.4.1` เอง |
| กรอกรหัส WiFi แล้วไม่เชื่อม | ตรวจรหัสว่าถูกต้อง หรือ WiFi นั้นบล็อก port 1883 ให้ลอง hotspot มือถือ Android |
| สถานะยังเป็น "รอสัญญาณ" หลัง 1 นาที | Device ID อาจผิด — ตรวจสอบ UUID ที่กรอกให้ตรงกับ URL ใน web app |
| Portal เปิดแล้วปิดเองใน 5 นาที | อุปกรณ์ restart อัตโนมัติ รอแล้วลองเชื่อมใหม่ |
